# Dataset Directory

Store datasets in the following layout:

```text
data/
  FHM/
    images/
      ...
    train.jsonl
    test.jsonl
  HarM/
    images/
      ...
    train.jsonl
    test.jsonl
  MAMI/
    images/
      ...
    train.jsonl
    test.jsonl
```

The dataset payload directories are ignored by Git. This repository tracks only this README and preprocessing scripts.
